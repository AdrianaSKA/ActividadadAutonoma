Información para la Página Web sobre el Document Object Model (DOM)

¿Qué es el DOM?

El Document Object Model (DOM) es una interfaz que permite representar documentos HTML y XML como una estructura jerárquica en forma de árbol. Gracias al DOM, los desarrolladores pueden acceder, manipular y modificar dinámicamente el contenido, la estructura y el estilo de una página web, haciendo posible la creación de sitios interactivos y dinámicos.

El Objeto document

El objeto document es el núcleo del DOM del lado del cliente. Proporciona acceso a los elementos de una página web a través de propiedades y métodos específicos.

Propiedades principales del objeto document:

  • document.body: Devuelve el elemento body del documento.
  • document.title: Retorna el título de la página web.
  • document.images: Es un arreglo que contiene todas las imágenes de la página.
  • document.bgColor: Permite definir o modificar el color de fondo del documento. El color puede indicarse con nombres en inglés ("red", "green") o valores hexadecimales (#FFFFFF para blanco).

Métodos comunes del objeto document:

  • open(): Abre el documento.
  • close(): Cierra el documento.
  • write() y writeln(): Escriben expresiones HTML en el documento.

Estructura del DOM: Nodos y Relaciones

El DOM está compuesto por nodos que representan los diferentes elementos del documento:

  • Element: Nodos que corresponden a etiquetas HTML como body, div o h1.
  • Text: Nodos que contienen el texto dentro de un elemento HTML.

Relaciones entre nodos:

  • Padre: Nodo superior inmediato.
  • Hijos: Nodos contenidos dentro de un nodo padre.
  • Hermanos: Nodos en el mismo nivel jerárquico.

Ejemplo de relaciones en el DOM: En una estructura como: 

<ul>
    <li id="miLI">Coffee</li>
    <li>Tea</li>
</ul>

Métodos para recorrer el DOM

El DOM proporciona métodos para navegar entre los nodos de un documento:

  • parentNode: Accede al nodo padre.
  • childNodes: Devuelve un arreglo con los nodos hijos.
  • firstChild: Accede al primer nodo hijo.
  • lastChild: Accede al último nodo hijo.
  • previousSibling: Navega al nodo hermano previo.
  • nextSibling: Navega al nodo hermano siguiente.

Ejemplo: Si tenemos el siguiente código HTML: 

<ul>
    <li id="miLI">Coffee</li>
    <li>Tea</li>
</ul>

Acceso y Modificación de Elementos HTML

Acceso a elementos:

  • getElementById(): Selecciona un elemento utilizando su atributo id.
  • getElementsByTagName(): Retorna un arreglo con todos los elementos que coinciden con una etiqueta específica.
  • getElementsByName(): Recupera elementos que tienen un atributo name igual al especificado.

Modificación de elementos:

  • innerHTML: Cambia el contenido HTML de un elemento.
  • setAttribute(atributo, valor): Modifica un atributo de un elemento.
  • Estilo y atributos: 
    elemento.atributo = valor;
elemento.style.propiedad = estilo;

Ejemplo práctico:

<button onclick="cambiarTexto()">Púlsame</button>
<p id="demo"></p>

<script>
function cambiarTexto() {
    document.getElementById("demo").innerHTML = "Hola Mundo";
}
</script>
